# fastapi-demo

## Dependencies

You will need:

- Python 3.6+

## Setup

To pull down this repo, run:

```
$ git clone git@github.com:lastnamevega/fastapi-demo.git && \
    cd fastapi-demo
```

Next, get your Python dependencies sorted:

```
$ ./setup && source venv/bin/activate
```

## Execution

Run the API webserver:

```
$ ./run
```

Run tests:

```
$ ./test
```

## Contributing

Autoformat the codebase:

```
$ ./format
```
